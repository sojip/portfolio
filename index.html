About SOJIP Franck

I often define myself as a telecommunications engineer with a passion for programming. I am one of those who learned to code thanks to the internet. After obtaining my degree, I was employed as a network engineer for a period during which I had various responsibilities, including network architecture, monitoring and maintenance, firewall configuration, systems administration, and cyber security. This is when I fell in love with Python because of its usefulness in cybersecurity, especially for malware analysis, vulnerability scanning, and penetration testing. After that, I decided to explore the language more deeply and took the CS50 online training on web programming with Python and Javascript. This course gave me a foundation with HTML5 - CSS3 - Javascript - JQuery - Bootstrap - Python - Flask - Git and allowed me to get another job opportunity later as a software developer. During this time, I was working on projects related to the automation of tasks and the digitization of the company's customer journey, like the functionnalities of the web portal for the company's customers. After finding myself to be a big fan of Javascript while working on a data visualization feature, I decided to follow The Odin Project's Full Stack Javascript path. This gave me the necessary skills to build and deploy full-stack web applications with the combination of ReactJs - Firebase - NodeJs - ExpressJs - MongoDB - Heroku. Javascript's ability to work on both the front and back end of a web application has made it my language of choice for specialization in web development, while my always learning mindset is still ready to discover the unknown.
