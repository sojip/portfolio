About Franck SOJIP

I often define myself as a network engineer with a passion for programming. I am one of those who learned to code thanks to the internet. After obtaining my degree, I was employed for a period during which I had various responsibilities, including network architecture, monitoring and maintenance, firewall configuration, systems administration, and cyber security. This is when I fell in love with programming due to Python's usefulness in cybersecurity, especially for malware analysis, vulnerability scanning, and penetration testing. At this moment, I decided to explore the language more deeply and took the CS50 online training on web programming with Python and Javascript. This course gave me a foundation with HTML5 - CSS3 - Javascript - JQuery - Bootstrap - Python - Flask - Git and allowed me to get another job opportunity later as a software developer. During this time, I worked on several projects related to the automation of tasks and the digitization of the company's customer journey, like the functionnalities of the user web portal. After finding myself being a big fan of Javascript while building a personal online marketplace project, I decided to follow The Odin Project's Full Stack Javascript path. This gave me the necessary skills to build and deploy full-stack web applications with the combination of ReactJs - Firebase - NodeJs - ExpressJs - MongoDB - Heroku. Javascript's ability to work on boths front and back end of a web application has made it my language of choice for specialization in web development, while my always learning mindset is still ready to discover the unknown.
